## Installation:

```bash
$ yarn add @mediba/stylelint-config-mediba --dev
```

## Usage:

Set your `stylelint` config.

```js
{
  "extends": "stylelint-config-mediba",
  "rules": {
    // If you customize rules, override here.
  }
}
```
